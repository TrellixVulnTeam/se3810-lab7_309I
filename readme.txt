Software Architecture - Group 4
Social AR App 
The model directory is sample code from TensorFlow: https://www.tensorflow.org

Set Up

For Windows:
1.- Install python version 3.6.8
2.- Run 'pip install tensorflow'
3.- On imageserver.py line 43, modify the absolute path for /model/tutorials/image/imagenet/classify_image.py
4.- Run imageserver.py
5.- Open https://localhost:25565
